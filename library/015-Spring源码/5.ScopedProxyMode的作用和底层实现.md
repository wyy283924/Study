## 5.ScopedProxyMode的作用和底层实现

要掌握ScopedProxyMode的底层实现，需要对Spring中的依赖注入、FactoryBean、ProxyFactory有一定的理解。

先考虑一种场景，有两个Bean，一个是WYYService，一个是RequestBean，代码如下

```java
@service
public class WYYService {
	@Resource
	private RequestBean requestBean; //RequestBean代理对象
    
	public String test() {
		return requestBean.test();
    }
}
```

```java
@Component
@Scope(value = "request",proxyMode = ScopedProxyMode.TARGET_CLASS)
public class RequestBean {
    
	public String test() { return "request...";}
    
}
```
