# 软件项目部署和访问问题
+ 把软件项目（网络应用程序）开发完成之后，需要让用户使用我们这个项目，但是目前基本上所有的软件项目都需要连接网络我们才能访问。为了让用户可以访问到我们项目，我们需要将项目部署到一个任何一个人都可以访问到计算机上即可（公网IP、局域网IP）。
+ 如果我们的应用程序想通过网络被别人访问到，需要借助一个软件将我们的项目设计成为一个可以通过网络访问的链接即可。
